# TLAPS A+ Content Modules — 14 modules cover all 213 listings

Built 2026-06-13. Instead of 210 unique A+ documents, you build **14 reusable modules**: one universal brand module + 13 category modules. Each listing's A+ page = **universal module (top) + its category module (below)**. Two modules per listing, assembled from 14 source images.

All images are 970×600 (Amazon standard full-width A+ module size), brand colors forest-green #1b3a1e + gold #b8965a + cream, built from the real product photos so they match what ships.

## How to use in Vendor Central A+ builder
1. Create A+ document → add a **Standard Image** module → upload `aplus_BRAND_all_categories.jpg` (the universal "everything TLAPS carries" block).
2. Add a second **Standard Image** module → upload the matching `aplus_<category>.jpg`.
3. Apply that A+ document to **every ASIN in that category** (one A+ doc per category, reused across all its listings). Apply to both I3ENT and I3ENU per the dual-vendor SOP.
